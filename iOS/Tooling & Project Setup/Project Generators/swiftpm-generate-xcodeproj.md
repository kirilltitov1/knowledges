# SwiftPM — generate-xcodeproj (устаревшее)

Apple удалил `swift package generate-xcodeproj` в пользу нативной интеграции Xcode с SPM.

## Текущее состояние
- Для iOS проектов рекомендуется использовать Xcode + SPM напрямую или генераторы (Tuist/XcodeGen).
- В монорепо/сложных кейсах — рассмотреть Bazel/rules_xcodeproj.
