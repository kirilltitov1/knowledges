# Buck — альтернативная билд‑система, генерация Xcode проекта

## Зачем
- Крупные кодовые базы, инкрементальные сборки, декларативные правила.

## Основы
- `BUCK` файлы описывают `apple_binary`, `apple_library`, `apple_bundle`.
- Генерация проекта: `buck project //...`.

## Плюсы / Минусы
- Плюсы: скорость, кэширование, инфраструктура для монорепо.
- Минусы: сложнее настройка, меньше комьюнити по iOS, утилиты вокруг.
