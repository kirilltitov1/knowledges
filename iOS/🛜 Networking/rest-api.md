# REST API


### HTTP Methods
- GET: Получение данных
- POST: Создание ресурса
- PUT/PATCH: Обновление
- DELETE: Удаление

### Request Components
- URL and endpoints
- Headers
- Query parameters
- Body (JSON, Form Data)
- Authentication

### Response Handling
- Status codes (200, 400, 500 series)
- Response headers
- Response body
- Error responses

### JSON Parsing
- JSONSerialization
- Codable protocol
- JSONEncoder/JSONDecoder
- Custom coding keys
- Date strategies
