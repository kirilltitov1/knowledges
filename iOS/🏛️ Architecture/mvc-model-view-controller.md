# MVC (Model-View-Controller)


### Концепция
- Model: Бизнес-логика и данные
- View: Отображение UI
- Controller: Координация между Model и View

### Реализация в iOS
- UIViewController как Controller
- Storyboards и XIBs
- Model objects
- Delegate pattern

### Проблемы
- Massive View Controller
- Tight coupling
- Сложность тестирования

### Когда использовать
- Простые приложения
- Прототипы
- Стандартный подход Apple
